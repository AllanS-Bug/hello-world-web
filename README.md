# Hello World Web Project

This is a simple web project that displays "Hello World" on a webpage using HTML, CSS, and JavaScript.

## Project Structure

```
hello-world-web
├── src
│   ├── index.html        # Main HTML document
│   ├── css
│   │   └── styles.css    # Styles for the webpage
│   └── js
│       └── script.js     # JavaScript code for DOM manipulation
├── .gitignore            # Files and directories to be ignored by Git
└── README.md             # Project documentation
```

## Getting Started

To run this project, follow these steps:

1. Clone the repository to your local machine.
2. Open the `src/index.html` file in your web browser.
3. You should see the "Hello World" message displayed on the page.

## Technologies Used

- HTML
- CSS3
- JavaScript
